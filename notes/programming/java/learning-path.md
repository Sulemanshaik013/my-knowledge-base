# Java Developer Learning Path

## 1. Java Fundamentals
- OOP principles (encapsulation, inheritance, polymorphism, abstraction)
- JVM, JRE, JDK architecture
- Memory management and garbage collection
- Basic syntax and data types
- Control flow and exception handling

## 2. Java Language Features
- Generics
- Collections Framework (Lists, Sets, Maps)
- I/O and NIO
- Lambda expressions and Streams API
- Concurrency basics (Threads, Executors)

## 3. Database Fundamentals
- SQL basics
- Database design principles
- JDBC fundamentals
- Connection pooling
- Transaction concepts

## 4. ORM Concepts
- Object-relational impedance mismatch
- JPA specification basics
- Entity lifecycle
- Basic ORM mapping concepts

## 5. Hibernate Core
- Configuration and setup
- Entity mapping with annotations
- Basic CRUD operations
- HQL/JPQL fundamentals
- Transaction management

## 6. Advanced Hibernate
- Relationships and associations
- Inheritance strategies
- Caching mechanisms
- Performance optimization
- Criteria API and native queries

## 7. Spring Core
- IoC and Dependency Injection
- Bean lifecycle
- Configuration approaches
- AOP fundamentals

## 8. Spring Data Access
- Transaction management
- Integration with Hibernate/JPA
- Exception handling
- Data repositories

## 9. Spring MVC & Web
- Web application architecture
- Controllers and request mapping
- View technologies
- REST APIs

## 10. Advanced Spring
- Spring Boot
- Spring Security
- Microservices concepts
- Testing Spring applications
